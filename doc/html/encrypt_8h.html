void offsetPpm ( std::fstream &  file)

Offsets ppm file to the right position.

Parameters
file- file stream
Returns
void

Function offsets file stream to the right position. It skips header of the file and stops at the first byte of the pixel array. It is used to make sure that we are writing to the pixel array position in ppm file and not damage the header.
